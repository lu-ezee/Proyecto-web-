# 🎧 Lucas Player

Aplicación de reproductor musical desarrollada en Android con Kotlin y Jetpack Compose.

---

## 🚀 Funcionalidades

- Reproducción de archivos MP3 locales
- Botón Play / Pause
- Botones de siguiente y anterior canción
- Cambio automático al finalizar una canción
- Interfaz simple centrada

---

## 🛠 Tecnologías

- Kotlin
- Android Studio
- Jetpack Compose
- MediaPlayer API

---

## 📌 Estado del proyecto

Versión funcional inicial del reproductor.  
Próximas mejoras: barra de progreso, diseño avanzado tipo Spotify.
